🛒 RETAIL_SALES_PREDICTION_USING_MACHINE_LEARNING
📘 About the Project

The RETAIL_SALES_PREDICTION_USING_MACHINE_LEARNING project is a data-driven solution designed to predict product sales across retail stores. Using data analysis, feature engineering, and machine learning algorithms, this project helps identify key factors influencing sales and supports better business decision-making.

It transforms raw retail data into meaningful insights and accurate predictions.

🚀 Features
📊 Data Preprocessing: Handles missing values and cleans raw data
📈 Exploratory Data Analysis (EDA): Visualizes trends and relationships
⚙️ Feature Engineering: Creates new meaningful features like outlet age
🤖 Machine Learning Models: Applies regression algorithms for prediction
📉 Model Evaluation: Compares models using performance metrics
🔍 Insight Generation: Identifies key factors affecting sales
⚙️ How It Works
📥 Data Input: Load dataset containing product and outlet details
🧹 Data Cleaning: Handle missing values and inconsistent data
🔄 Data Transformation: Encode categorical variables
📊 Feature Engineering: Create new useful features
🤖 Model Training: Train regression models (Linear, Decision Tree, Random Forest)
📏 Evaluation: Measure model performance using metrics
📤 Prediction: Generate sales predictions
📊 Dataset
Records: 8,523
Features: 12

Includes:
